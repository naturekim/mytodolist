# naturekim TODO LIST  

VanilaJS로 만든 심플한 Todo List입니다.  
[사이트 바로가기](https://nature-todolist.netlify.app/)
<br />

## Features
* Todo 목록 추가 및 삭제
* 진행상태 표시: 전체 개수 및 완료 개수, 진행률 바
* 오늘 날짜 표시
* 제목 수정
* 피드백 작성 및 수정
* LocalStorage에 데이터 저장

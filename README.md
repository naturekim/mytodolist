# naturekim TODO LIST  

VanilaJS로 만든 심플한 Todo List입니다.  
## Features
* LocalStorage에 데이터 저장
* 목록 추가 및 삭제 기능
* 목록 전체 개수 및 완료 개수, 진행률 바 표시
* 오늘 날짜 표시
* 제목 수정 기능
* 피드백 작성 기능
